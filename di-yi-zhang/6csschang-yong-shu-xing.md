###6.1.1background-color 设置元素背景色
```
<style>
    /* 给.box的背景色设置为红色 */
    .box {
        background-color: red;
    }
</style>
```
> * 使用 background-color 属性为元素设置背景色，其默认值是 transparent 也就是“透明”的意思，可使用颜色名称、RGB、RGBA、十六进制数来指定颜色(例如#ffff等)。
